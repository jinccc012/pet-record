# Pet Record

寵物紀錄網站，用來紀錄狗狗與貓貓的基本資料、日常生活紀錄與健康紀錄。

本專案包含：

- React + Vite + TypeScript 前端
- Spring Boot 後端
- Neon PostgreSQL 資料庫
- Cloudflare R2 檔案儲存
- Signed URL 檔案直傳
- Oracle Cloud Always Free VM 部署後端
- Cloudflare Pages 部署前端
